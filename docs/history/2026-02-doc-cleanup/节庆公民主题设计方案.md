# 节庆公民主题设计方案

## 概述

基于"节庆公民"风格，我为议起读网站重新设计了一套完整的前端界面。这个设计将议事规则学习转化为一场充满活力的公民节庆活动。

## 设计特点

### 视觉风格
- **高饱和色彩**：主色 #be123c (深红)，次色 #e11d48 (亮红)，强调色 #2563eb (蓝色)
- **节庆氛围**：淡粉红背景 (#fff1f2)，营造温暖节庆感
- **海报式排版**：大胆的标题字体 (Impact) 和戏剧性布局

### 交互体验
- **脉冲动画**：按钮和重要元素采用脉冲效果，增强参与感
- **戏剧性动效**：浮动、弹跳、发光等动画营造节庆氛围
- **玻璃态导航**：现代感的半透明导航栏

## 实现的功能

### 1. 主题切换系统
- 在首页右上角添加了主题选择器
- 支持"经典政务"和"节庆公民"两种主题
- 主题选择会保存到 localStorage，下次访问时自动应用

### 2. 节庆公民主题样式
- 完整的CSS变量系统，支持主题切换
- 专门的节庆组件样式 (fc-* 类名)
- 响应式设计，适配各种屏幕尺寸

### 3. 重新设计的首页
- 根据选择的主题动态调整内容和样式
- 节庆主题下的特殊元素：节庆徽章、浮动装饰、特殊图标
- 保持功能完整性的同时提供截然不同的视觉体验

## 使用方法

1. **主题切换**：点击首页右上角的"主题"按钮，选择想要的主题风格
2. **自动保存**：选择的主题会自动保存，下次访问时生效
3. **响应式**：两种主题都完全支持移动端和桌面端

## 技术特性

- **CSS变量系统**：使用 `data-theme` 属性控制主题切换
- **TypeScript支持**：完整的类型定义和类型安全
- **性能优化**：CSS-only动画，避免JavaScript性能开销
- **无障碍支持**：支持 `prefers-reduced-motion` 媒体查询

## 设计理念

**节庆公民**主题将严肃的议事规则学习转化为一场充满活力的公民节庆。通过：

- **情感调性**：热情洋溢、积极向上、充满行动力
- **视觉语言**：节庆海报的大胆排版 + 公民运动的激情色彩
- **交互体验**：脉冲式反馈营造参与感和紧迫感

让用户在学习议事规则的过程中感受到参与公民活动的热情和使命感。

## 下一步扩展

这个主题系统为未来扩展奠定了基础：

1. **更多主题**：可以轻松添加更多主题变体
2. **个性化设置**：用户可以自定义颜色和字体
3. **季节性主题**：根据节日或活动推出特殊主题
4. **组织定制**：为不同组织提供专属主题

通过这个设计，议起读网站现在拥有了两种截然不同但都专业的视觉风格，能够满足不同用户群体和使用场景的需求。
